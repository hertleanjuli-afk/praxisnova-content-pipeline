# PraxisNova AI - Content-Pipeline

Ein Text rein, 5 Formate raus. KI-gesteuerte Content-Erstellung fuer LinkedIn, Facebook, Newsletter und mehr.

## Setup

```bash
npm install
npm run dev
```

## Deployment auf Vercel

1. GitHub Repo erstellen und Code pushen
2. Auf vercel.com importieren (Framework: Next.js)
3. Environment Variable setzen: `GEMINI_API_KEY` = dein Key
4. Deploy

## Tech Stack

- Next.js 14 (App Router)
- TypeScript + Tailwind CSS
- Gemini 2.5 Flash API (Free Tier)
- Gemini 2.0 Flash Image (Free Tier)
